## Strong matches
### continue-dev ships "Focus Mode" — single-file context for deep-dive sessions
**Source:** [Continue.dev Blog](https://blog.continue.dev/focus-mode)
**What it is:** Continue's VS Code extension now lets you pin a single file as the entire context window, suppressing workspace-wide indexing noise while you debug one gnarly module.
**Why it fits:** Context-window hygiene is one of the core AuDHD dev struggles — too much ambient signal triggers the "everything is equally important" spiral; Focus Mode is a direct mechanical fix.
**Possible angle:** Walk through how restricting context is actually a cognitive-load win, and why neurotypical devs might not reach for this feature as instinctively.

### Pieces for Developers adds "Cognitive Snapshot" — auto-annotates a work session on pause
**Source:** [Pieces Blog](https://pieces.app/blog/cognitive-snapshot)
**What it is:** When you step away from a coding session, Pieces now captures the open files, recent clipboard history, and last shell commands into a named snapshot you can restore later.
**Why it fits:** Context-switching back into a half-finished task is disproportionately expensive with ADHD — losing the mental thread means re-reading everything; this externalises the "where was I?" problem.
**Possible angle:** Frame it as an executive-dysfunction prosthetic: the snapshot does the working-memory job your brain refuses to queue overnight.
